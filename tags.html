Goosander Cottage Home page

Listing some of the important tags in HTML5. Also shows how to make tables using table.

Block Elements

Tag Name Purpose
h1-h6 heading include a heading
p paragraph include paragraphs
table table includes a table
tr table row includes a table row
th table header includes a table header
td table data includes a table data cell

Inline Elements

Tag Name Purpose Example Result
em emphasised Make the text in italics <em>text</em> text
a link Make a link <a href="index.html">text</a> text
strong strong Make the text strong <strong>text</strong> text
img image Make an image <img src="image.png" width="50%" height="50%">
